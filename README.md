## About The Project

A Main component of minecraft is crafting, with this plugin i want to transform minecraft into an idle game

Here's how:
* You can Place multiblocks
* Multiblocks allow the user to automate crafting
* Selling crafted items gains money for the user
* With money the player can upgrade machines

## Roadmap

- [x] Make a database Connection class
- [x] Make a core that allows the programm to access the data from the database
- [x] Make a abstract class for the multiblocks that has functionality for placing, removing, and opening the inventory
- [ ] Give the gui buttons that allow the user to remove and upgrade the machine
- [ ] Give the gui buttons that allow the user to select available nearby multiblocks that it can interact with.
- [ ] Allow the user to input items in the storageInterface
- [ ] Allow the robot arm to take items from the storageInterface
- [ ] Allow the user to set a recipe on the crafting table
- [ ] Allow the robot arm to put items on the crafting table
- [ ] Make the crafting table craft an item if all recipe items are available
- [ ] Make the crafting table output the item on the output field when the item is crafted
- [ ] Make the output area behave like a storage interface

See the [open issues](https://github.com/othneildrew/Best-README-Template/issues) for a full list of proposed features (and known issues).
